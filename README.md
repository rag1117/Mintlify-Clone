# Mintlify Landing Page

A recreation of the Mintlify homepage featuring the hero, company logos, and footer sections.

##Live Link:
https://my-mintlify-five.vercel.app/

## Sections Recreated

### 1. Hero Section (`.hero`)
- **Navbar** — Sticky navigation with logo, nav links (Resources, Documentation, Customer, Blog, Pricing), and CTA buttons (Contact sales, Start for free)
- **Hero content** — “NEW” pill badge, main headline (“The Intelligent Knowledge Platform”), supporting copy, email input + “Start now” CTA, and hero preview image

### 2. Company Logos Section (`.mid-logos`)
- Grid of customer/partner logos (Anthropic, Coinbase, Microsoft, Perplexity, HubSpot, X, PayPal, Lovable)
- 4-column responsive layout on a dark background

### 3. Footer (`.site-footer`)
- **Top row** — Mintlify logo and social links (LinkedIn, X, GitHub)
- **Navigation** — Columns: Explore, Resources, Documentation, Company, Legal
- **Security badge** — “Backed By Enterprise-Grade Security” with AICPA SOC II badge
- **Bottom row** — System status (“All systems normal”), copyright (© 2026 Mintlify, Inc.), theme toggle (System / Light / Dark)
